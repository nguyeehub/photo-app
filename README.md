# Photo App

A native desktop photo management app built with Tauri, React, and TypeScript. Work in progress.

## What it does

Browse, preview, and organise photos from local folders and external devices. Features include burst photo grouping, marquee selection, and a sidebar for navigating directories.

## Getting started

Prerequisites: [Node.js](https://nodejs.org/), [Rust](https://www.rust-lang.org/tools/install), and the [Tauri prerequisites](https://v2.tauri.app/start/prerequisites/) for your platform.

```sh
npm install
npm run tauri dev
```
